Videos/Intro.gif
            ^ add yourgif.gif at here
            ^ drag & drop it onto convertgiftopngset.exe

This will convert a gif into a set of pngs at:
Videos/Intro/Files/
and move the .gif to Videos/Intro/Intro.gif

Source code of the executable is the .py script.
You can compile that by yourself, running these commands in cmd (from current directory):
` pip install pillow pyinstaller ` <- install dependencies
` pyinstaller --onefile --noconsole convertgiftopngset.py ` <- compile the script
